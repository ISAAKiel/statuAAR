# SchmidtEtAl2007: statuaar table example

A measurements table from *Schmidt et al. 2007*. Raw data, stature
estimates and bone measures from European Neolithic and Early Bronce Age
archaeological context.

## Format

A data frame with 10 rows and 7 variables.

- **skeleton** ID count.

- **period:** German: Schnurkeramik (corded ware), Glockenbecherkultur
  (Bellbeaker Culture), Bandkeramik (Linear Pottery Culture),
  Aunjetitzkultur (Unetice Culture).

- **sex:** weiblich (female), männlich (male)

- **stature:** stature, Fully & Pineau (1960) (mm)

- **Fem1:** Femur M1 measures (mm)

- **Tib1a:** Tibia M1a measures (mm)

- **Hum1:** Humerus M1 measures (mm)

## References

Schmidt K, Bruchhaus H, Bindl R (2007). “Körperhöhenschätzung an
ausgewählten neolithischen und bronzezeitlichen Skeletten.”
*Archäologische Informationen*, **30,1**, 51–69.
[2026-03-10](https://isaakiel.github.io/statuAAR/reference/2026-03-10).

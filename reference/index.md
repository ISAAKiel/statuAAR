# Package index

## Basic functions

- [`getFormulaDataframe()`](https://isaakiel.github.io/statuAAR/reference/getFormulaDataframe.md)
  : Returns a data.frame containing formula informations.
- [`getFormulaMeasures()`](https://isaakiel.github.io/statuAAR/reference/getFormulaMeasures.md)
  : Returns the measures for the formula abbreviation
- [`getStature()`](https://isaakiel.github.io/statuAAR/reference/getStature.md)
  : Returns a list with data.frames with stature estimations.
- [`getStatureDataframe()`](https://isaakiel.github.io/statuAAR/reference/getStatureDataframe.md)
  : Returns the list of stature estimates from getStature() as
  data.frame.

# Returns a list with data.frames with stature estimations.

Returns a list with data.frames of stature estimations for each
specified formula and the data provided in a data.frame of class
statuaar_data_table.

## Usage

``` r
getStature(shortnames, statuaar_data_table)
```

## Arguments

- shortnames:

  A vector of short names of the formula, e.g. c('bb65', 'tg01').

- statuaar_data_table:

  A data.frame of class statuaar_data_table as provided by
  prep.statuaar.data(). \#'

  - **Ind**: identifyer for each individual.

  - **Sex**: sex of the individual. Accepted values: 1 (male), 2
    (female), 3 (indet) or `m`(ale), `f`(emale), `indet`.

  - **variable**: short name of the measure

  - **value**: measured value.

## Value

A list with a data.frame for each specified stature estimation formula.
The item name corresponds to the specified formula. Each data.frame with
seven columns:

- **ind:** individual identifyer (rownames),

- **sex:** as provided for calculation: m, f, indet.

- **stature:** estimated on the provided sex and bone measures,

- **bone measure name(s)**: bones used for calculation,

- **female stature**: columns with alternative stature for three sex
  classes,

- **male stature**,

- **indet. stature** and

- **n_measures**: number of bone measures included: e.g. 2 Fem2 (left,
  right) + 1 Tib1

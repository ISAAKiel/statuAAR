# statuaar_formula: List of implemented formula for stature estimation

statuaar_formula: List of implemented formula for stature estimation

## Format

A list with a list for each formula providing three items.

- **short:** character, abbreviation, e.g. bb65.

- **long:** character, full name used for the function, e.g.
  breitinger_bach_1965.

- **measures:** character, vector of short measure names, e.g. Hum1.

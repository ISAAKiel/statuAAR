# measures.concordance.rollet: concordance for Rollet (1888) measure names to list of valid measures according to Martin (1928)

measures.concordance.rollet: concordance for Rollet (1888) measure names
to list of valid measures according to Martin (1928)

## Format

A data frame with 44 rows and 3 variables.

- **short:** abbreviation, e.g. Fem1l

- **long:** full name, e.g. Femur.1.left

- **own:** column for the own measure names

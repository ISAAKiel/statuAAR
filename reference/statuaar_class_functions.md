# checks and sets statuaar objects

The statuaar_data_table is the central data structure for stature
estimation in `statuAAR` package. It is a data.frame with specific
variables, s.
[`statuAAR::prep.statuaar.data`](https://isaakiel.github.io/statuAAR/reference/readuserdata.md).

## Usage

``` r
as.statuaar_data_table(x, ...)

as.statuaar_statistics(x, ...)

is.statuaar_data_table(x, ...)

is.statuaar_statistics(x, ...)
```

# dl.rollet1888: statuaar data table

A measures data table (list form) from *Rollet 1888, pp. 16-23*.

## Format

A data frame with 100 rows and 18 variables.

- **Ind** identifier

- **Sex:** f, m, i -\> female, male, indet.

- **Variable** bone measure

- **value** measures mm
